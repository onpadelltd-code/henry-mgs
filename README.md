# Henry @ MGS — Family Hub

Family planner + Owl Academy (adaptive learning app) for Henry, Year 4 at MGS Junior School.

- `/` — family planner (Today / Week / Month / Manage), PIN-protected
- `/owl/` — Owl Academy training missions, PIN-protected

Note: `api.php` (email import backend) does not run on GitHub Pages — the Import tab needs separate PHP hosting.
